1.1 주요 특징

AXGATE는 IPv4/IPv6 네트워크 운영환경에서 고성능 하드웨어 기반의 시스템으로 방화벽, IPS/DDoS, VPN, Anti-Spam, Anti-Virus, 어플리케이션등의 주요 보안 기능을 UTM(Unified Threat Management)장비 형태로 제공합니다. 본 장비는 Gigabit 트래픽까지 성능 저하 없이 패킷을 처리할 수 있도록 개발된 우수한 성능의 하드웨어 일체형 통합 보안 장비로서 시스템을 보호하고 보안 사고를 예방합니다. AXGATE는 외부 네트워크와 내부 네트워크 경계 단에 인라인(In-line) 형태로 설치되어 인가된 보안관리자가 정의한 보안정책에 따라 AXGATE를 거치는 모든 패킷의 내용 및 행동을 네트워크 성능 저하 없이 실시간으로 분석합니다. 또한 하나의 H/W박스에서 방화벽, 가상사설망, IPS/DDoS 보안기능을 통합 구현함으로써 일관되고 통합적인 보안정책 관리가 가능하게 되어 관리비용 절감으로 인한 운영 측면 효율성과 하나의 장비운용으로 인한 에너지 절감 등 비용절감 효율성을 동시에 제공합니다. AXGATE가 보호하고자 하는 네트워크와 인터넷 등 외부 네트워크 사이에서 유통하는 정보 흐름을 관리자가 설정한 정책에 따라 통제합니다. AXGATE의 주요 기능은 다음과 같습니다.

  1. 사용자 기반 접근 제어
    • 기존 방화벽은 IP 주소 정보만을 기반으로 접근을 제어했습니다. 그러나 AXGATE는 IP 주소 정보와 더불어 사용자정보를 기반으로 접근을 제어할 수 있으며, 보다 효율적인 보안 인증 기능을 지원합니다.
    • 사용자 계정 (ID + Password) 정보를 통해 접근 제어
    • 사용자 그룹별, 개인별로 구분하여 보안 정책을 차등 적용 가능
    • 사용자 단말에 Agent를 따로 설치할 필요 없이 Web을 통한 보안 인증 지원
    • 사용자 별 이용 트래픽, 공격 유형 등의 통계 지원
    • 신뢰성 있는 3rd Party 인증을 통한 접근을 활용하기 위해 Radius, LDAP, AD, MS-SQL, ORACLE, Local DB등과의 연동 지원
    • 고정 IP환경과 더불어 유동 IP를 사용하는 환경에서도 사용자 기반의 접근 제어 가능

  2. Zone 기반 접근 제어
  3. 논리적 가상화 방화벽
  4. IPS
  5. 어플리케이션 제어
  6. 개인정보보정책
